# Дашборд тривог по 20 громадах (лютий–червень, помісячно)

Накопичувальна система: збирає тривоги по вашому списку громад з alerts.in.ua,
складає у локальну базу й показує рейтинг, помісячне порівняння та типи загроз.

## Чому накопичувально

API alerts.in.ua віддає історію **лише за останній місяць** (period=month_ago).
Старі місяці заднім числом недоступні. Тому:

- `collect_hromady.py` запускається регулярно і **докладає** свіжі дані в базу
  (без дублів — дедуплікація за id тривоги). Так база росте вперед.
- `import_history.py` додає **історичне вивантаження** (лютий–квітень), яке ви
  отримаєте від власника даних (air-alarms.in.ua / alerts.in.ua за запитом).
- Обидва потоки лягають в одну базу — дашборд бачить повну історію.

## Файли

```
hromady_config.py      # ВАШ список 20 громад (uid + назва + область)
collect_hromady.py     # накопичувальний збір з API -> data/alerts.db
import_history.py      # імпорт історичного CSV -> data/alerts.db
build_stats.py         # база -> data/stats.json (для дашборду)
index.html             # дашборд
data/alerts.db         # база (зараз — демо-дані)
data/stats.json        # агрегат для дашборду
```

## Крок 1. Впишіть свої громади

Відкрийте `hromady_config.py`. Зараз там 20 демо-громад — замініть їх своїми.
Для кожної потрібен **uid** (числовий ідентифікатор у API), а не лише назва.
Повний офіційний список локацій:
https://docs.google.com/spreadsheets/d/1XnTOzcPHd1LZUrarR1Fk43FUyl8Ae6a6M7pcwDRjNdA/edit

Знайдіть громаду в таблиці, візьміть uid, впишіть. (Можу допомогти зіставити —
просто надішліть мені назви громад.)

## Крок 2. Отримайте API-токен (безкоштовно)

https://alerts.in.ua/api-request → токен у змінну середовища:
```bash
export ALERTS_TOKEN="ваш_ключ"
```

## Крок 3. Зберіть дані

```bash
python3 collect_hromady.py     # ~10 хв для 20 громад (ліміт 2 запити/хв)
python3 build_stats.py         # формує stats.json
```

Історичні місяці (якщо отримали вивантаження):
```bash
python3 import_history.py лютий_квітень.csv
python3 build_stats.py
```

## Крок 4. Подивіться дашборд

```bash
python3 -m http.server 8000    # http://localhost:8000
```

Натисніть на громаду в рейтингу — розкриється деталізація: метрики,
помісячний графік і розбивка за типами загроз.

## Регулярне оновлення

Раз на місяць достатньо (бо API і так віддає лише місячне вікно):
```
0 5 1 * * cd /шлях && ALERTS_TOKEN=ключ python3 collect_hromady.py && python3 build_stats.py
```
Або через GitHub Actions — див. `.github/workflows/update.yml` (додайте секрет ALERTS_TOKEN).

## Застереження

- **Старі місяці**: лютий–квітень доступні лише через імпорт вивантаження.
  Через API заднім числом їх отримати неможливо.
- **Типи загроз**: API розрізняє повітряну тривогу, артобстріл, вуличні бої,
  хімічну й ядерну загрозу. Поділу «ракети / дрони» в структурованих даних
  немає — лише інколи в текстовому полі notes.
- **Тривалість громад орієнтовна**: частина тривог має прогнозований
  (а не реальний) час відбою — це поле `calculated` / показник «прогноз. час».
- Для публічного сервісу alerts.in.ua вимагає проксувати запити через ваш сервер.
